# Changelog

## [1.0.1-beta] - 2026-01-04

### Fixed
- Исправлена ошибка форматирования при недостаточном балансе
- Исправлено поведение для недоступных тарифов (показывается стандартное меню)
- Исправлено поведение для запрещённых групп тарифов (показывается стандартное меню)

### Changed
- Убрана проверка баланса из модуля (делегируется стандартному обработчику)
- Модуль теперь перенаправляет на стандартное меню подтверждения с опциями

### Removed
- Удалён код проверки баланса (~40 строк)
- Удалены неиспользуемые импорты: ceil, InlineKeyboardBuilder, get_balance, MAIN_MENU, PAYMENT, edit_or_send_message
- Удалена функция register_module_middleware()
- Удалены сообщения о недостаточном балансе из texts.py

### Optimization
- Сокращено ~31% кода (с 320 до 226 строк)
- Упрощена логика в handle_quick_renew

## [1.0.0-alpha] - 2026-01-03

### Added
- Кнопка быстрого продления текущего тарифа
- Автоматическая проверка баланса перед продлением
- Кнопка "Показать все тарифы" для доступа к полному списку
- Middleware для перехвата callback renew_key
- Обработка недоступных тарифов
